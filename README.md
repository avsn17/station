# ✦ study.girl

> a terminal-aesthetic study stationery tool with your companion **Mochi** 🎀

![study.girl screenshot](https://img.shields.io/badge/version-3.0-ff85c8?style=flat-square&labelColor=fde8f3)
![GitHub Pages](https://img.shields.io/badge/deploy-GitHub_Pages-c9a7ff?style=flat-square&labelColor=f2ecfc)

## features

- **✦ boot sequence** — animated startup with system checks
- **♡ mochi** — interactive SVG mascot companion who reacts to everything you do
- **✦ notes** — write, tag & save study notes (export to .txt)
- **◷ timer** — pomodoro / focus timer with circular progress ring
- **✧ flashcards** — flip cards with 3D animation, shuffle, navigate
- **✿ tasks** — study checklist with satisfying checkoffs
- **♡ mood tracker** — log how you're feeling with mochi's empathetic responses
- **✦ quote bar** — rotating study motivation quotes
- **✧ sparkles** — floating ambient sparkle background

## deploy to GitHub Pages

1. fork or clone this repo
2. go to **Settings → Pages**
3. set source to **Deploy from branch → main → / (root)**
4. your page will be live at `https://yourusername.github.io/study-girl`

## local use

just open `index.html` in any browser — no build step, no dependencies.  
all data persists in `localStorage`.
